## 公有属性
| 属性名称   | 描述     | 属性类型 | 取值范围 | 设置 | 上报 |
| -------- | -------- | ------ | ------- | ---- | ---- |
| SoftVersion | 软件版本，格式为1.0 | String  | 5字节 | 0 | 1 |
| HardVersion | 硬件版本，格式为1.0 | String  | 5字节    | 0 | 1 |
| ECBSoftVersion | 电控板软件版本，格式为1.0 | String  | 5字节 | 0 | 1 |
| Reboot | 重启 | null | null | 1 | 0 |
| Reset | 恢复出厂设置 | null | null | 1 | 0 |
